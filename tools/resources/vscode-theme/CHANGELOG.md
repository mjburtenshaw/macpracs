# Change Log

All notable changes to the "Themodynamic Stark" theme will be documented in this file.

## [1.0.0] - 2026-01-28

### Added
- Initial release
- Themodynamic Dark variant
- Themodynamic Light variant
- Full syntax highlighting for:
  - JavaScript/TypeScript
  - Python
  - HTML/CSS
  - JSON
  - Markdown
  - Git diffs
- Terminal color support
- UI theming for all VS Code interface elements
- Character-themed color palette from the Kuroki universe

### Design Philosophy
- Void Band (3 colors): Backgrounds, shadows, deep tones
- Key Band (3 colors): Paper base, highlights, light features
- Memory Band (24 colors): Character-themed chromatic colors
  - Memory.Whispers (red): Alice's favorite color
  - Memory.Wrath (orange): Colt's eye striations
  - Memory.Sage (green): Alice's eyes
  - Memory.Silver (blue): Silver's eyes
  - Memory.Garden (yellow): Mary Feybrook's hair
  - Memory.Grace (purple): Colt's scarf

### Color Philosophy
"Color is rare. When it appears, it means something." - Pops of meaningful color against an otherwise desaturated world.
