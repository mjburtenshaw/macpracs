# Themodynamic Stark - Slack Themes

> *"The system does not seek peace. It seeks containment."*

Slack color themes based on the Themodynamic Stark palette from the Kuroki universe.

## Installation

1. Open Slack
2. Go to **Preferences** (Cmd+,)
3. Click **Themes** in the sidebar
4. Scroll down to **Create a custom theme**
5. Copy one of the color codes below
6. Paste into the text field
7. Click **Done**

## Customization

Slack themes use 10 hex codes separated by commas in this order:

1. **Column Background** - Left sidebar background
2. **Menu Background Hover** - Hover state for channels
3. **Active Item** - Selected/active channel background
4. **Active Item Text** - Text color for selected channel
5. **Hover Item** - Hover background color
6. **Text Color** - Main text color in sidebar
7. **Active Presence** - Online status indicator color
8. **Mention Badge** - Background for mention badges
9. **Top Nav Background** - Top navigation bar background
10. **Top Nav Text** - Top navigation text color

### Example: Custom Orange Accent

To use Memory.Wrath (Colt's eyes) as the accent:

```
#131114,#201d21,#ad6c45,#edebe4,#2f2c30,#edebe4,#92b282,#e09970,#0f0e10,#849ab8
```

Change positions 3 and 8 to your preferred accent color from the palette.

## Troubleshooting

### Colors don't look right

1. Make sure you copied the entire code including all commas
2. Check that there are exactly 10 hex codes
3. Try refreshing Slack (Cmd+R)

### Theme looks different on mobile

Slack mobile apps may render colors differently. The theme is optimized for desktop.

### Want to switch back

Slack has built-in themes you can select:
- Preferences → Themes → Choose a preset
