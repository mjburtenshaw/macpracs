# Quick Install - Slack Themes

## Themodynamic Dark

```
#131114,#201d21,#7a2656,#edebe4,#2f2c30,#edebe4,#92b282,#ad7676,#0f0e10,#849ab8
```

**To install:**
1. Copy the code above
2. Open Slack → Preferences (Cmd+,)
3. Click Themes → Create a custom theme
4. Paste the code
5. Click Done
